# OLS Mathematical Derivation

> Ordinary Least Squares (OLS) gives us a **closed-form solution** — a direct formula to compute the best slope and intercept without any iteration.

---

## The Setup

In Simple Linear Regression we have:
- One input variable X (e.g., CGPA)
- One output variable Y (e.g., Package)
- Goal: Find the **best** values of `m` (slope) and `b` (intercept) for the line `y = mx + b`

---

## Two Approaches to Finding Parameters

| Aspect | Closed-Form (OLS) | Non Closed-Form (Gradient Descent) |
|---|---|---|
| Method | Direct formula application | Iterative optimization |
| Mathematical Basis | Algebraic solution using derivatives | Optimization using calculus iteratively |
| Computation | Single calculation | Multiple iterations |
| Best For | Low-dimensional data | High-dimensional data |
| sklearn Class | `LinearRegression` | `SGDRegressor` |
| Speed (Low Dims) | Very fast | Slower |
| Speed (High Dims) | Can be slow / infeasible | Efficient |

### What is a Closed-Form Solution?

A **closed-form solution** is a mathematical expression that can be evaluated in a finite number of standard operations (addition, subtraction, multiplication, division) — no iteration needed.

Example: the quadratic formula is closed-form:
```
x = (-b ± √(b² - 4ac)) / 2a
```

---

## Understanding the Best Fit Line

The **Best Fit Line** is the specific line where:
- The sum of all squared errors is minimized
- Every point may deviate, but the total squared deviation is the smallest possible

| Line Type | Error Level |
|---|---|
| Random line | High error |
| Good line | Medium error |
| Best Fit Line | **Minimum error** |

---

## Deriving the Error (Loss) Function

### Step 1 — Error for a Single Point

For any data point `i`:
- Actual value: `yᵢ`
- Predicted value: `ŷᵢ = mxᵢ + b`
- Residual: `dᵢ = yᵢ - ŷᵢ = yᵢ - mxᵢ - b`

### Step 2 — Why Not Sum Errors Directly?

```
E_total = d₁ + d₂ + ... + dₙ
```

**Problem:** Positive and negative errors cancel out — the sum could be zero even for a terrible line.

### Step 3 — Squared Errors

Square each error to prevent cancellation:

```
E = Σ dᵢ² = Σ(yᵢ - ŷᵢ)²
```

### Step 4 — Complete Loss Function

Substituting `ŷᵢ = mxᵢ + b`:

```
E(m, b) = Σ(yᵢ - mxᵢ - b)²
```

This is the **Loss Function** (also called Cost Function or SSE).

---

## Why Squared Errors — Not Absolute?

| Criterion | Squared Error (d²) | Absolute Error (|d|) |
|---|---|---|
| Cancellation | Prevents cancellation | Prevents cancellation |
| Penalizes Outliers | Heavily penalizes large errors | Linear penalty |
| Differentiability | Smooth, differentiable **everywhere** | Not differentiable at zero |
| Mathematical Convenience | Easy to differentiate | Harder to work with |
| Graph Shape | Smooth parabola | V-shape with sharp corner |

<Callout type="warning">
The absolute value function has a sharp corner at x = 0 — it's continuous but **not differentiable** at zero. This makes finding the minimum mathematically difficult and incompatible with gradient-based optimization.
</Callout>

**Why not d⁴ or d⁶?**
If d² solves the problem (prevents cancellation AND is differentiable), higher powers only add unnecessary computational cost and over-penalize outliers.

---

## Error Function Geometry

### E as a Function of m (b fixed)

When plotting E vs m, varying the slope creates a **parabola**:
- Too small m → line too flat → high error
- Optimal m → line fits well → minimum error
- Too large m → line too steep → high error

### E as a Function of b (m fixed)

Similarly, plotting E vs b also creates a **parabola**.

### Combined: 3D Surface

When both m and b vary, E(m, b) forms a **3D bowl-shaped surface (convex function)**:

| Region | m, b Values | Error | Description |
|---|---|---|---|
| Mountain peaks | Extreme values | Very high | Poor fit lines |
| Slopes | Near-optimal | Medium | Improving fit |
| Valley bottom | Optimal m, b | **Minimum** | Best fit line |

<Callout type="success">
The surface is **convex** — it has exactly ONE minimum (global minimum). No local minima to get stuck in. We can find the exact bottom using calculus.
</Callout>

## Mathematical Derivation Using Partial Derivatives

At the minimum of E(m, b), both partial derivatives must be zero:

```
∂E/∂m = 0   and   ∂E/∂b = 0
```

---

### Deriving the Formula for b

**Step 1:** Start with the loss function:
```
E = Σ(yᵢ - mxᵢ - b)²
```

**Step 2:** Take partial derivative with respect to b:
```
∂E/∂b = Σ 2(yᵢ - mxᵢ - b) · (-1)
       = -2 Σ(yᵢ - mxᵢ - b)
```

**Step 3:** Set equal to zero:
```
-2 Σ(yᵢ - mxᵢ - b) = 0
   Σ(yᵢ - mxᵢ - b) = 0
```

**Step 4:** Expand the summation:
```
Σyᵢ - m Σxᵢ - nb = 0
```

**Step 5:** Divide by n:
```
ȳ - mx̄ - b = 0
```

**Step 6:** Solve for b:

```
b = ȳ - m × x̄
```

---

### Deriving the Formula for m

**Step 1:** Start with the loss function.

**Step 2:** Take partial derivative with respect to m (chain rule):
```
∂E/∂m = -2 Σ xᵢ(yᵢ - mxᵢ - b)
```

**Step 3:** Set equal to zero:
```
Σ xᵢ(yᵢ - mxᵢ - b) = 0
```

**Step 4:** Substitute `b = ȳ - mx̄`:
```
Σ xᵢ(yᵢ - mxᵢ - ȳ + mx̄) = 0
Σ xᵢ[(yᵢ - ȳ) - m(xᵢ - x̄)] = 0
```

**Step 5:** Expand:
```
Σ xᵢ(yᵢ - ȳ) - m Σ xᵢ(xᵢ - x̄) = 0
```

**Step 6:** Use the property `Σ(xᵢ - x̄) = 0` to simplify:
```
Σ xᵢ(yᵢ - ȳ) = Σ(xᵢ - x̄)(yᵢ - ȳ)
Σ xᵢ(xᵢ - x̄) = Σ(xᵢ - x̄)²
```

**Step 7:** Solve for m:

```
m = Σ(xᵢ - x̄)(yᵢ - ȳ) / Σ(xᵢ - x̄)²
```

---

## Final OLS Formulas

| Parameter | Formula | Order |
|---|---|---|
| Slope (m) | `m = Σ(xᵢ - x̄)(yᵢ - ȳ) / Σ(xᵢ - x̄)²` | Calculate first |
| Intercept (b) | `b = ȳ - m × x̄` | Uses calculated m |

### Intuitive Understanding

**Slope m:**
- Numerator: measures how X and Y vary together ≈ covariance of X and Y
- Denominator: measures how X varies by itself ≈ variance of X
- Essentially: "How much does Y change when X changes?"

**Intercept b:**
- The best fit line always passes through the point `(x̄, ȳ)`
- From point-slope form: `ȳ = mx̄ + b` → `b = ȳ - mx̄`

### Properties of the Best Fit Line

| Property | Mathematical Expression | Meaning |
|---|---|---|
| Passes through means | `(x̄, ȳ)` is on the line | Line goes through center of data |
| Sum of residuals = 0 | `Σ(yᵢ - ŷᵢ) = 0` | Positive and negative errors balance |
| Minimizes SSE | Minimum of `Σ(yᵢ - ŷᵢ)²` | Smallest possible squared error |

## Python Implementation from Scratch

```python
class MeraLR:
    """
    Custom Linear Regression using OLS formulas.
    Attributes:
        m (float): Slope of the regression line
        b (float): Y-intercept of the regression line
    """

    def __init__(self):
        self.m = None
        self.b = None

    def fit(self, X_train, y_train):
        """Train using OLS formulas."""
        num = 0
        den = 0
        for i in range(X_train.shape[0]):
            num += (X_train[i] - X_train.mean()) * (y_train[i] - y_train.mean())
            den += (X_train[i] - X_train.mean()) ** 2

        self.m = num / den
        self.b = y_train.mean() - (self.m * X_train.mean())

        print(f"Slope (m): {self.m}")
        print(f"Intercept (b): {self.b}")

    def predict(self, X_test):
        """Predict using ŷ = mx + b"""
        return self.m * X_test + self.b
```

### Formula to Code Mapping

| Formula Component | Code |
|---|---|
| `xᵢ - x̄` | `X_train[i] - X_train.mean()` |
| `yᵢ - ȳ` | `y_train[i] - y_train.mean()` |
| `(xᵢ - x̄)(yᵢ - ȳ)` | `(X_train[i] - X_train.mean()) * (y_train[i] - y_train.mean())` |
| `(xᵢ - x̄)²` | `(X_train[i] - X_train.mean()) ** 2` |
| `m = num/den` | `self.m = num / den` |
| `b = ȳ - mx̄` | `self.b = y_train.mean() - (self.m * X_train.mean())` |

## Complete Working Example

```python
import numpy as np
import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.linear_model import LinearRegression

# Load data
df = pd.read_csv('placement.csv')
X = df.iloc[:, 0].values  # CGPA
y = df.iloc[:, 1].values  # Package

X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.2, random_state=2
)

# Custom model
lr = MeraLR()
lr.fit(X_train, y_train)

# Verify against sklearn
sklearn_lr = LinearRegression()
sklearn_lr.fit(X_train.reshape(-1, 1), y_train)

print("Comparison of Parameters:")
print(f"Our m:      {lr.m}")
print(f"sklearn m:  {sklearn_lr.coef_[0]}")
print(f"Our b:      {lr.b}")
print(f"sklearn b:  {sklearn_lr.intercept_}")
```

## Key Formulas Summary

| Formula | Expression | Purpose |
|---|---|---|
| Line Equation | `y = mx + b` | The model being fitted |
| Loss Function | `E = Σ(yᵢ - mxᵢ - b)²` | What we minimize |
| Slope (m) | `Σ(xᵢ - x̄)(yᵢ - ȳ) / Σ(xᵢ - x̄)²` | Rate of change |
| Intercept (b) | `b = ȳ - mx̄` | Y-value when X = 0 |
| Prediction | `ŷ = mx + b` | Predict new values |

---

## Important Concepts

| Concept | Explanation |
|---|---|
| Best Fit Line | The line that minimizes total squared error |
| Residual | Difference between actual and predicted: `yᵢ - ŷᵢ` |
| OLS | Ordinary Least Squares — minimizes sum of squared residuals |
| Closed-Form | Direct formula without iteration |
| Partial Derivative | Derivative with respect to one variable, holding others constant |

---

## Limitations

| Limitation | Explanation | Solution |
|---|---|---|
| Only Simple LR | Works for single feature only | Extend to Multiple Linear Regression |
| No Regularization | Doesn't handle overfitting | Add L1/L2 regularization |
| Assumes Linearity | Data must have linear relationship | Use polynomial features if needed |
| Sensitive to Outliers | Squared errors amplify outliers | Use robust regression methods |
