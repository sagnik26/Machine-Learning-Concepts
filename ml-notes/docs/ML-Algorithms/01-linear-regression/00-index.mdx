## Core Formulas

### Simple Linear Regression

| Formula | Expression |
|---|---|
| Prediction | `ŷ = mx + b` |
| Slope | `m = Σ(xᵢ - x̄)(yᵢ - ȳ) / Σ(xᵢ - x̄)²` |
| Intercept | `b = ȳ - mx̄` |
| Loss (SSE) | `E = Σ(yᵢ - ŷᵢ)²` |

### Multiple Linear Regression

| Formula | Expression |
|---|---|
| Prediction | `ŷ = β₀ + β₁x₁ + ... + βₘxₘ` |
| Matrix Form | `Ŷ = Xβ` |
| Normal Equation | `β = (XᵀX)⁻¹ Xᵀy` |

### Evaluation Metrics

| Metric | Formula |
|---|---|
| MAE | `(1/n) Σ ‖yᵢ - ŷᵢ‖` |
| MSE | `(1/n) Σ (yᵢ - ŷᵢ)²` |
| RMSE | `√MSE` |
| R² | `1 - Σ(yᵢ - ŷᵢ)² / Σ(yᵢ - ȳ)²` |
| Adjusted R² | `1 - (1 - R²) · (n-1)/(n-1-k)` |

## Choosing the Right Approach

| Scenario | Recommended Approach | sklearn Class |
|---|---|---|
| 1 feature | Simple LR with OLS | `LinearRegression` |
| 2–10,000 features | Multiple LR with OLS | `LinearRegression` |
| 10,000+ features | Multiple LR with Gradient Descent | `SGDRegressor` |
| Non-linear data | Polynomial Regression | `PolynomialFeatures` + `LinearRegression` |
| Overfitting present | Ridge or Lasso Regression | `Ridge`, `Lasso` |

## Quick Start (sklearn)

```python
from sklearn.linear_model import LinearRegression
from sklearn.model_selection import train_test_split
from sklearn.metrics import r2_score
import numpy as np

# Simple Linear Regression
X = df[['cgpa']].values   # 2D
y = df['package'].values

X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.2, random_state=2
)

model = LinearRegression()
model.fit(X_train, y_train)
y_pred = model.predict(X_test)

print(f"Slope:     {model.coef_[0]:.4f}")
print(f"Intercept: {model.intercept_:.4f}")
print(f"R²:        {r2_score(y_test, y_pred):.4f}")
```
