Multiple Linear Regression

Introduction​

Multiple Linear Regression (MLR) extends Simple Linear Regression to handle multiple independent variables. It models the linear relationship between several features (X₁, X₂, X₃, …) and a single continuous target variable (y).

Regression vs Classification​

AspectRegressionClassification
Output TypeContinuous numerical valuesDiscrete categories / labels
ExamplesPrice prediction, temperature forecastSpam detection, image classification
EvaluationMAE, MSE, R², RMSEAccuracy, Precision, Recall, F1
AlgorithmsLinear Regression, Ridge, LassoLogistic Regression, SVM, Decision Trees

Types of Linear Regression​

TypeFeaturesEquationDimensionsUse Case
Simple LR1y = mx + b2D (Line)Single predictor analysis
Multiple LR2+y = β₀ + Σ βᵢxᵢ3D+ (Hyperplane)Multiple predictors
Polynomial1+ (with powers)y = β₀ + β₁x + β₂x²2D (Curve)Non-linear relationships

Mathematical Foundation​

General Form​

Multiple Linear Regression extends the equation to multiple independent variables:

y = β₀ + β₁x₁ + β₂x₂ + β₃x₃ + ... + βₙxₙ

Compact Notation:

y = β₀ + Σᵢ₌₁ⁿ βᵢxᵢ
SymbolMeaning
yPredicted target value
β₀Intercept (bias term)
βᵢCoefficient for feature xᵢ
xᵢIndependent variable (feature i)
nNumber of features

Matrix Form​

For multiple samples and features:

Y = Xβ + ε
SymbolDescriptionDimensions
YTarget vector(m × 1)
XFeature matrix (with intercept column)(m × (n+1))
βCoefficient vector((n+1) × 1)
εError vector(m × 1)

Model Representation​

From Simple to Multiple LR​

Simple Linear Regression (2D):

  • 1 variable: x, y
  • Equation: y = mx + b
  • Geometry: a line in 2D space
  • Example: CGPA → Package

Multiple Linear Regression (3D, 2 features):

  • 2 variables: x₁, x₂, y
  • Equation: y = β₀ + β₁x₁ + β₂x₂
  • Geometry: a plane in 3D space
  • Example: CGPA + IQ → Package

General (nD):

  • n features
  • Equation: y = β₀ + β₁x₁ + ... + βₙxₙ
  • Geometry: a hyperplane in (n+1)-dimensional space

Concrete Example (2 features)​

Package = β₀ + β₁ × CGPA + β₂ × IQ

Python Implementation​

Generating Synthetic Data​

from sklearn.datasets import make_regression
import pandas as pd
import numpy as np

X, y = make_regression(
    n_samples=100,
    n_features=2,
    n_informative=2,
    n_targets=1,
    noise=50
)

df = pd.DataFrame({
    'feature1': X[:, 0],
    'feature2': X[:, 1],
    'target': y
})

Train / Test Split​

from sklearn.model_selection import train_test_split

X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.2, random_state=3
)
# Training: 80 samples | Testing: 20 samples

Model Training​

from sklearn.linear_model import LinearRegression

lr = LinearRegression()
lr.fit(X_train, y_train)
y_pred = lr.predict(X_test)

print(f"Coefficients: {lr.coef_}")
print(f"Intercept:    {lr.intercept_}")

Output:

Coefficients: [59.64518074, 13.20409431]
Intercept:    -7.55491251398082

Resulting Equation:

y = -7.555 + 59.645 × x₁ + 13.204 × x₂

Parameter Interpretation​

ParameterValueMeaning
β₀ (intercept)-7.555Baseline value when all features = 0
β₁ (feature1)59.645Target increases by 59.645 per unit increase in feature1
β₂ (feature2)13.204Target increases by 13.204 per unit increase in feature2
Warning

Coefficient comparisons for feature importance are only valid if features are on the same scale. Always normalize / standardize features before comparing coefficients.

Model Evaluation​

from sklearn.metrics import mean_absolute_error, mean_squared_error, r2_score
import numpy as np

mae  = mean_absolute_error(y_test, y_pred)
mse  = mean_squared_error(y_test, y_pred)
rmse = np.sqrt(mse)
r2   = r2_score(y_test, y_pred)

print(f"MAE:  {mae:.3f}")
print(f"MSE:  {mse:.3f}")
print(f"RMSE: {rmse:.3f}")
print(f"R²:   {r2:.4f}")

Assumptions of Linear Regression​

AssumptionDescriptionHow to Check
LinearityLinear relationship between features and targetResidual plots
IndependenceObservations are independentDomain knowledge, Durbin-Watson test
HomoscedasticityConstant variance of residualsResidual vs. fitted plot
NormalityResiduals are normally distributedQ-Q plot, histogram
No MulticollinearityFeatures are not highly correlatedVIF (Variance Inflation Factor)

Cost Function and Minimization​

Objective: Find β that minimizes:

J(β) = (1/2m) Σ(yᵢ - ŷᵢ)²

Step-by-Step:

  1. Define hypothesis: ŷᵢ = β₀ + β₁xᵢ₁ + β₂xᵢ₂
  2. Compute residuals: eᵢ = yᵢ - ŷᵢ
  3. Square residuals: eᵢ² = (yᵢ - ŷᵢ)²
  4. Sum all squared residuals: SSE = Σ eᵢ²
  5. Normalize: J(β) = SSE / (2m)
  6. Take partial derivatives and set to zero: ∂J/∂βⱼ = 0 for all j
  7. Solve the normal equations: XᵀXβ = Xᵀy
  8. Final solution: β = (XᵀX)⁻¹ Xᵀy

Prediction Example​

For a new data point (x₁* = 1.0, x₂* = 0.5):

ŷ* = -7.555 + 59.645 × 1.0 + 13.204 × 0.5
   = -7.555 + 59.645 + 6.602
   = 58.692

When to Use Multiple Linear Regression​

Use when:

  • Relationship between variables is approximately linear
  • Target variable is continuous
  • Multiple features influence the outcome
  • Interpretability is important
  • Need a quick baseline model

Avoid when:

  • Relationship is highly non-linear → use polynomial/tree-based models
  • Target is categorical → use classification algorithms
  • Features are highly correlated → use regularization (Ridge/Lasso)
  • Small dataset with many features → risk of overfitting

Feature Importance from Coefficients​

From trained coefficients [59.645, 13.204]:

  • Feature 1: |β₁| = 59.645 → higher magnitude → more impact
  • Feature 2: |β₂| = 13.204 → lower magnitude → less impact

Model Selection Criteria​

CriterionPreferTrade-off
Fewer FeaturesSimple, interpretable modelMay miss important patterns
More FeaturesBetter fit, lower training errorRisk of overfitting
OptimalBalance via regularization (Ridge/Lasso)Complexity vs. performance
