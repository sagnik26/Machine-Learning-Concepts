Core Formulas​

Simple Linear Regression​

FormulaExpression
Predictionŷ = mx + b
Slopem = Σ(xᵢ - x̄)(yᵢ - ȳ) / Σ(xᵢ - x̄)²
Interceptb = ȳ - mx̄
Loss (SSE)E = Σ(yᵢ - ŷᵢ)²

Multiple Linear Regression​

FormulaExpression
Predictionŷ = β₀ + β₁x₁ + ... + βₘxₘ
Matrix FormŶ = Xβ
Normal Equationβ = (XᵀX)⁻¹ Xᵀy

Evaluation Metrics​

MetricFormula
MAE(1/n) Σ ‖yᵢ - ŷᵢ‖
MSE(1/n) Σ (yᵢ - ŷᵢ)²
RMSE√MSE
R²1 - Σ(yᵢ - ŷᵢ)² / Σ(yᵢ - ȳ)²
Adjusted R²1 - (1 - R²) · (n-1)/(n-1-k)

Choosing the Right Approach​

ScenarioRecommended Approachsklearn Class
1 featureSimple LR with OLSLinearRegression
2–10,000 featuresMultiple LR with OLSLinearRegression
10,000+ featuresMultiple LR with Gradient DescentSGDRegressor
Non-linear dataPolynomial RegressionPolynomialFeatures + LinearRegression
Overfitting presentRidge or Lasso RegressionRidge, Lasso

Quick Start (sklearn)​

from sklearn.linear_model import LinearRegression
from sklearn.model_selection import train_test_split
from sklearn.metrics import r2_score
import numpy as np

# Simple Linear Regression
X = df[['cgpa']].values   # 2D
y = df['package'].values

X_train, X_test, y_train, y_test = train_test_split(
    X, y, test_size=0.2, random_state=2
)

model = LinearRegression()
model.fit(X_train, y_train)
y_pred = model.predict(X_test)

print(f"Slope:     {model.coef_[0]:.4f}")
print(f"Intercept: {model.intercept_:.4f}")
print(f"R²:        {r2_score(y_test, y_pred):.4f}")
